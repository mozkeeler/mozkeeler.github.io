History Lessons

This is a quick follow-up to my previous post. As a possible corrolary to Cunningham’s Law, it seems the best way to learn about the past on the Internet is to write an uninformed blog post and have colleagues on Twitter fill you in. Apparently using #ifdef DEBUG_<person> was common and even supported by default by the build system (it would grab your login name and automatically define DEBUG_\<login>). There’s a mozilla.dev.platform discussion about removing it here and two related bugs for anyone interested in the historical background.
